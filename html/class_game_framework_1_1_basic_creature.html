Detailed Description

Represents a basic creature in the game world, capable of attacking, defending, and interacting with lootable objects.

This class provides fundamental behaviors for a creature, including handling attacks, receiving damage, and looting items. It extends the Creature class and implements the IBasicCreature interface.

Processes an incoming hit by reducing the hit points based on the defense value.

The actual damage applied is reduced by the defense value, if available. The hit points are then decreased by the resulting damage, but will not drop below zero.
